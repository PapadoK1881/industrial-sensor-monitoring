# Σύστημα Παρακολούθησης Βιομηχανικών Αισθητήρων

## Περιγραφή
Η εργασία αφορά την ανάπτυξη μιας απλής web εφαρμογής για την παρακολούθηση βιομηχανικών αισθητήρων, όπως θερμοκρασία, υγρασία και κραδασμοί.
Ο χρήστης μπορεί να βλέπει τους αισθητήρες, να καταγράφει μετρήσεις και να εμφανίζονται ειδοποιήσεις όταν οι τιμές ξεπερνούν κάποια όρια.

## Εκτέλεση
Για να τρέξει η εφαρμογή:

1. Εγκατάσταση XAMPP  
2. Τοποθέτηση του project στον φάκελο htdocs  
3. Δημιουργία της βάσης δεδομένων μέσω phpMyAdmin  
4. Εκκίνηση Apache και MySQL  
5. Άνοιγμα στον browser στη διεύθυνση:
   http://localhost/industrial_sensor_project
